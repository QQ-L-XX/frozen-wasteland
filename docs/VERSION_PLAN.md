# 极寒末世版本计划 / Frozen Wasteland Version Plan

更新时间 / Updated: 2026-06-30

## 中文计划

| 版本 | 定位 | 主要目标 | 验收标准 | 当前状态 |
|---|---|---|---|---|
| `v1.0.0-candidate` | GitHub 试玩候选版 | 核心循环可跑，仓库页面完整，内置 QA 能验证 P0 | TypeScript 检查通过；P0 一周烟测通过；README/发布说明/仓库 About 完成 | 已发布 |
| `v1.0.0` | 稳定试玩版 | 完成 Day 1-7 人工长测，补真实截图，确认存档/读档/搜刮/撤离不破局 | Cocos 预览 Day 1-7 长测通过；至少 2-4 张截图；无 P0 阻塞 bug | 进行中 |
| `v1.1.0` | 中期经营版 | 强化世界网络、外部哨站和生产链，让 Day 10-40 有稳定目标 | 至少 3 条生产链有明确收益；哨站维护/补给有清晰反馈；中期资源不空转 | 计划中 |
| `v1.2.0` | 角色与事件版 | 补幸存者个人线、关系事件、长期灾变和挑战剧本 | 至少 6 条角色/关系事件链；至少 3 个长期灾变；至少 3 个挑战剧本 | 计划中 |
| `v1.3.0` | 表现打磨版 | 补截图、音效、视觉反馈、仓库展示和试玩包 | 基础音效覆盖主流程；README 使用真实截图；可导出试玩包 | 计划中 |
| `v2.0.0` | 长线完整版 | 面向 80-100 小时内容量，补终局路线和可重复目标 | 每 10-20 天有新目标/风险；多结局路线可追踪；长线内容量显著扩展 | 远期 |

## 当前优先级

1. `v1.0.0`：Cocos 预览人工长测 Day 1 到 Day 7。
2. `v1.0.0`：补真实截图，替换 README 的临时 SVG 主视觉或增加截图区。
3. `v1.0.0`：确认许可证策略。
4. `v1.1.0`：扩展中期生产链与世界网络节奏。
5. `v1.2.0`：补角色线、灾变和挑战剧本。

## 已接入游戏内

- 系统菜单新增 `📋 版本` 面板。
- 面板显示当前版本、下一版本、GitHub 地址、各版本状态、目标和验收标准。
- 后续每完成一个版本节点，会同步提交并推送到 GitHub；正式版本再打 tag。

## English Plan

|---|---|---|---|---|
