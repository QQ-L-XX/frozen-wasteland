# 截图目录 / Screenshot Directory

真实游戏截图建议放在本目录，README 会优先展示这些文件。

Recommended screenshot files:

| File | 中文说明 | English |
|---|---|---|
| `01-title-or-start.png` | 标题或开局界面 | Title or new-game screen |
| `02-base-hud.png` | 基地 HUD 和建造区 | Base HUD and build area |
| `03-scavenge-map.png` | 搜刮地图 | Scavenging map |
| `04-completion-panel.png` | v1.0 完成度面板 | v1.0 completion panel |

截图要求 / Screenshot notes:

- 使用 Cocos Creator 预览窗口截图。
- 保持 HUD、主要交互和画面主体清晰。
- 不要包含浏览器隐私信息、账户信息或本地路径。
- PNG 优先，宽度建议 1280px 或以上。

- Capture from the Cocos Creator preview window.
- Keep HUD, main interactions, and gameplay content readable.
- Do not include private browser/account data or local paths.
- Prefer PNG files, ideally 1280px wide or larger.
