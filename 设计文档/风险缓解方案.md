# 风险缓解方案

> 针对游戏性评审中 3 个核心风险 | 2026-05-18

---

## 风险 1：搜刮模板不够 → 重复疲劳

### 问题
每种建筑只有 3 个模板 → 第 10-15 天后重复。

### 缓解

#### A. 模板 × 模块交换 = 百级变体
```
基础布局(3个骨架) × 模块交换(每个房间2-3变体) × 随机种子
= 有效变体 100+ 种

以医院为例：
  骨架A/B/C → 走廊直通/曲折/堵塞 → 病房满/空/改 → 药房有锁/无锁/破坏
  → 不是换皮，是结构真的不同
```

#### B. 区域类型差异化机制
不同区域不只是贴图不同，而是**搜刮规则不同**：

| 区域 | 独特机制 |
|------|---------|
| 公寓 | 垂直探索（楼梯/电梯井） |
| 商业街 | 半开放，车辆残骸可搜刮 |
| 工业区 | 大型设备可拆解 → 稀有零件 |
| 地铁 | 黑暗需照明，线性+分支选择 |
| 军事区 | 先破解防御再搜刮 |
| 研究中心 | BOSS 战斗+解谜驱动 |

→ 不同区域 = 不同玩法，不只是不同贴图

#### C. 枯竭度自然轮换
```
郊区枯竭 → 逼迫去商业街 → 商业街枯竭 → 去医院 → ...
自然引导玩家接触不同类型，而非在同一区域反复刷
```

#### D. 动态事件注入新鲜感
同一区域第二次进入时：
- 上次没有掠夺者，这次有了
- 上次走廊畅通，这次被雪埋了
- 上次故事种子没触发，这次触发了

## 风险 2：早期 10 天可能劝退

### 问题
硬核玩家享受压力，休闲玩家可能第三天就弃。

### 缓解

#### A. 简单模式再松一步
```
当前简单模式：不死人、搜刮窗口×1.5、暴风雪×0.6
补充：
  · 初始食物多给 3 天量（30→45 单位）
  · 第一天不弹食物警告，改为第 2 天
  · 第一次搜刮时老兵说"这边很安全，慢慢搜"
```

#### B. 留一天安全期
```
Day 1 上午：加燃料（教学）
Day 1 下午：建木墙（教学）
Day 1 晚上：小人对话提一句"明天该找吃的了"
Day 2 上午：弹出每日报告 + 世界地图
→ 玩家有一个完整日熟悉界面，不被立刻逼迫
```

#### C. 失败是故事，不是 Game Over
```
如果新手冻死：
  不是弹出 GAME OVER → 重来

  而是弹出：
  "你的基地在第三天熄灭了。老兵最后的日记写道：也许我们该早点出门。
   [再来一次] [降低难度] [查看统计]"

→ 环世界式：失败也是叙事，降低挫败感
```

## 风险 3：三层切换不连贯

### 问题
基地↔世界地图↔搜刮切换太频繁或太久 → 打断心流。

### 缓解

#### A. 减少切换频率（设计层面）
```
预期节奏：每 2-3 游戏天出门一次，不是每天
建造等待时给进度条+预计时间，让玩家知道"还有 30 秒"
```

#### B. 动画掩盖载入（技术层面）
```
基地↔世界地图：缩放过渡 0.5s，无载入
世界地图→搜刮：雪地行军动画+旁白 1-2s
搜刮→基地：总结面板即是过渡
目标：室内 < 1s，开放 < 1.5s
```

#### C. 搜刮回来后即时反馈
```
搜刮结束回到基地时：
  · 搜到蓝图 → 建造列表对应项高亮闪烁
  · 搜到大量物资 → 仓库图标数字弹出动画
  · 队员受伤 → 医务室自动高亮
  · 发现故事种子 → 世界地图按钮闪烁
→ 切换有"收获感"而非"加载感"
```

#### D. 全局快捷键
```
搜刮场景中按 Tab → 半透明显示基地状态摘要
（食物：35 · 燃料：2天 · 建造中：木墙 67%）
→ 不需要切回基地就能知道全局状态
```
