## 第二篇：无线电交互

### 2.1 无线电室建造

```
建造需求：
  · 金属 ×10
  · 电子元件 ×5
  · 无线电蓝图（搜刮工业区/军事区获得）
  · 温度 ≥ 0°C（可在较冷房间运作）

建造后无线电室默认为"被动接收"模式。
需要指派幸存者（研究优先级）来操作。
```

### 2.2 两种模式

```
模式 A：被动接收（持续开机）

  效果：
    · 随机收到信号（求救、新区域标记、特殊事件）
    · 消耗电力（如果基地有发电机）或消耗燃料（手动发电）
    · 信号频率约 1 次 / 2-5 天
    · 持续开机增加被掠夺者定位的概率（+5%/天）

  不需要玩家主动操作。信号来了弹出通知即可。


模式 B：主动扫描

  玩家点击无线电 → 打开交互面板：

  ┌──────────────────────────────────┐
  │ 📡 无线电 —— 主动扫描              │
  │                                  │
  │  调频旋钮  ┌────●────┐            │
  │            │ 86.7 MHz│            │
  │            └─────────┘            │
  │                                  │
  │  [扫描当前频段]  ← 消耗 1 电子元件  │
  │                                  │
  │  ▸ 上次扫描结果：                  │
  │    发现新区域：北区工业园（距离 3km） │
  │                                  │
  │  ▸ 已知信号：                     │
  │    · 86.7 MHz — 来源不明（第5天发现）│
  │    · 102.3 MHz — 南侧避难所       │
  │                                  │
  │  [关闭]                          │
  └──────────────────────────────────┘
```

### 2.3 信号类型

| 信号类型 | 频率 | 内容 |
|----------|------|------|
| 求救信号 | 不定 | 有人被困 → 可选救援（道德抉择） |
| 区域信号 | 固定频段 | 发现新区域地图标记 |
| 军用频道 | 102-110 MHz | 军用蓝图提示、特殊装备位置 |
| 商业频道 | 88-95 MHz | 商队路线信息、贸易机会 |
| 未知信号 | 不定 | 故事种子——可能指向结局"信号回应" |

### 2.4 无线电升级

| 升级 | 需求 | 效果 |
|------|------|------|
| 信号增强器 | 电子元件×3 | 主动扫描范围 +50% |
| 自动扫描 | 电子元件×5+蓝图 | 被动模式也能发现新区域（无需操作员） |
| 跨区域通讯 | 稀有蓝图+电子元件×10 | 可与其他幸存者聚落建立常驻通讯 |
