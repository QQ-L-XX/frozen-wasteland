# 温室与种植系统

> 填补 P1 缺失项 | 2026-05-18 | v0.1

---

## 一、定位

温室是基地从"纯消耗"转向"自给自足"的分水岭。建温室前，食物全靠搜刮和狩猎；建温室后，食物有了可预测的稳定来源。

---

## 二、温室建造

### 2.1 建造需求

| 需求 | 数量 | 来源 |
|------|------|------|
| 木材 | 10 | 搜刮/平整雪地 |
| 玻璃/薄膜 | 5 | 搜刮商场、住宅 |
| 土壤 | 3 单位 | 挖掘冻土后处理 |
| 温室蓝图 | 1 | 搜刮获得（商业街、学校概率高） |

### 2.2 温室运作前提

```
温室不是建好就自动产食物。需要：
  · 温度 ≥ +2°C（必须有供暖管道接入）
  · 有水（雨水收集器 / 化雪水）
  · 有种子（搜刮获得第一批，后续可自留种）
  · 有光照（基础日光 / 工坊升级→人工光源）
  · 有幸存者分配到"种植"工作
```

### 留种机制

```
收获时可以选择：
  · 全部收获 → 100% 食物产出，但需要新种子才能继续种
  · 留种 50% → 50% 食物产出，自动获得 1 颗新种子（无需再搜刮）
  
  策略：第一茬全部收获（缓解饥饿），第二茬开始留种（建立自循环）
```

## 五、温室升级路径

| 等级 | 建造需求 | 效果 |
|------|---------|------|
| 基础温室 | 木材10+玻璃5+土壤3+蓝图 | 4 格种植空间，基础光照（日光） |
| 加温温室 | +金属5+管道接入 | 温度下限 -5°C（-3°C 也能种耐寒作物） |
| 人工光照 | +电子元件3+工坊升级 | 光照不受暴风雪影响，生长速度 +20% |
| 水培系统 | +电子元件5+金属10+蓝图 | 无需土壤，生长速度 +30%，可种高级作物 |
| 自动化温室 | +稀有蓝图+高智力解读 | AI 自动管理浇水/收获，不需要种植者 |

## 六、气候与季节影响

```
核冬天背景下没有传统四季，但有以下周期：

  阴天周期（Perlin Noise，约 3-6 天一次）：
    自然光照降低 → 基础温室生长速度 ×0.7
    人工光照温室不受影响

  暴风雪：
    基础温室光照 = 0（完全遮光）→ 生长暂停
    人工光照温室继续生长

  极寒夜（<-40°C）：
    基础温室可能温度跌破 +2°C → 作物冻死
    加温温室可应对
```

## 七、作物疾病与虫害

```
风险事件（低概率，事件系统驱动）：

  虫害（5%/季）：指定一格作物被虫蛀 → 产出 -50%
    → 应对：需要农药（搜刮获得）或手动清除（消耗工时）

  枯萎（3%/季）：如果温度波动超过 10°C/天 → 随机一格作物枯萎
    → 应对：保持稳定供暖

  病菌（2%/季）：在连作同一作物超过 3 茬后触发
    → 应对：轮作不同作物（不同格种不同作物）
```

## 八、产出平衡参考

```
一个 4 格基础温室（种耐寒土豆，有基础光照）：

  每天产出 = 4 格 × (4 单位/5天) = 3.2 单位食物/天
  单人格消耗 = 2 单位/天（基本）~ 4 单位/天（重劳动+寒冷）

  → 一个基础温室可以养活 1-1.5 个幸存者
  → 要养活 4 人基地，需要 3 个基础温室或 1 个升级温室
  → 自给自足的临界点约在 Day 12-15（建好 2-3 个温室后）
```

---

> 温室的核心设计意图：不是"建了就不用管"，而是持续消耗管理精力（温度、水、光照、虫害），但有持续的可预测回报。和搜刮的无序暴富形成对比——温室让你睡得安稳，但不会让你一夜暴富。
