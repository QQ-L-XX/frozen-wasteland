# 搜刮中的 AI 队友行为设计

> 填补 P0 缺失项 · 关联「搜刮场景流程设计」「幸存者 AI 行为树」| 2026-05-18

---

## 一、核心问题

玩家在搜刮场景里带 1-2 个 AI 队友。这些队友在基地里有一套完整的自主 AI，但进入搜刮场景后——规则变了。搜刮场景是快节奏、高压力、有倒计时的，不能用"慢慢扫描工作优先级"那一套。

### 设计原则
- **跟随为主，指令为辅**：AI 队友默认跟随玩家，等待指令。不自作主张去搜刮（以免触发噪音）。
- **战斗自主**：敌人出现时，AI 队友自动进入战斗，无需玩家微操。
- **指令轻量**：右键点击下达简单命令（搜刮这个 / 守在这里 / 去那边），不打断节奏。

---

## 二、默认跟随行为

### 2.1 跟随编队

```
玩家（Leader）始终可控。

AI 队员默认状态：跟随

编队形态：
  · 玩家在前 / 队员在后 2-3 格
  · 偏移 ±1 格，避免完全重叠
  · 玩家转向时队员同步转向
  · 移动速度 = min(队长速度, 队员速度) → 保持一致
```

### 2.2 跟随中的自主反应

即使处于"跟随"状态，AI 队友也会对以下情况自主反应：

| 触发条件 | 反应 |
|----------|------|
| 敌人进入视野 | 自动进入战斗（详见第四节） |
| 队员自身血量 < 30% | 走到玩家身后 2 格，停止攻击 |
| 发现隐藏物资点 | 头顶 💡 气泡提示，不自行搜刮 |
| 队友倒地 | 如果带医疗包 → 自动施救 |
| 玩家倒地 | 如果自己血量 > 50% → 优先施救玩家 |

## 三、指令系统

### 3.1 指令类型

玩家右键点击场景中的物体或地面 → 弹出上下文指令菜单：

```
点击物资点：
┌─────────────┐
│ 🔍 搜刮这里  │ ← AI 队员走过去执行搜刮
│ 🛡️ 守住这里  │ ← 队员在此位置警戒
│ [取消]      │
└─────────────┘

点击敌人：
┌─────────────┐
│ ⚔️ 攻击目标  │ ← 优先攻击此敌人
│ [取消]      │
└─────────────┘

点击地面：
┌─────────────┐
│ 🚶 移动到这  │ ← 队员走到该位置
│ 🛡️ 守住这里  │ ← 在此位置警戒
│ [取消]      │
└─────────────┘

点击玩家自己（或快捷键 F）：
  → 所有队员回到跟随状态
```

### 3.2 指令的快捷操作

为减少右键菜单打断节奏，提供快捷操作：

| 快捷键 | 效果 |
|--------|------|
| `Tab` | 切换到下一个队员（玩家直接控制该队员，之前的玩家角色变成 AI 跟随） |
| `F` | 所有队员回到跟随 |
| `1/2/3` | 选择对应编号的队员（选中的队员会高亮，下一次指令针对选中队员） |
| 双击物资点 | 最近的非玩家队员自动搜刮（安静模式） |
| `G` | 全员守住当前位置 |

## 四、战斗行为

### 4.1 战斗自主程度

AI 队员的战斗是**高自主、低微操**的：

```
战斗中的 AI 队员：

  ├─ 自动索敌：优先攻击离自己最近的敌人
  │   └─ 除非玩家标记了优先目标（右键→攻击目标）
  │
  ├─ 自动走位：
  │   ├─ 远程武器队员：保持距离（敌我间距 4-8 格），自动后撤
  │   ├─ 近战武器队员：正面迎敌，不躲避
  │   └─ 掩体意识：如果 3 格内有掩体 → 概率 60% 移动过去
  │
  ├─ 自动换弹：弹药耗尽自动换弹（换弹时间 2 秒，期间无法攻击）
  │
  ├─ 撤退阈值：
  │   ├─ 血量 < 40% → 走向玩家（寻求掩护）
  │   ├─ 血量 < 20% → 主动后退，只在安全时攻击
  │   └─ 所有队员血量 < 30% → 弹出建议："是否撤离？"
  │
  └─ 使用道具：
      ├─ 自身血量 < 40% AND 有医疗包 → 自行治疗（3 秒，期间不攻击）
      └─ 队友倒地 AND 有医疗包 → 优先施救
```

### 4.2 战斗中的玩家控制

```
如果玩家希望手动指挥战斗：

  【选项 1：切换控制】
  Tab 切换到该队员 → 玩家直接控制
  → 原玩家角色变成 AI 跟随 + 自主战斗
  → 可随时 Tab 切回来

  【选项 2：指令覆盖】
  右键点击敌人 → AI 队员优先攻击那个目标
  → 覆盖"优先攻击最近的"默认规则
  → 持续直到目标死亡或玩家说"回到跟随"
```

## 五、AI 队员的搜刮行为

### 5.1 玩家指令搜刮

玩家指令→搜刮时，AI 队员的行为：

```
指令："搜刮这个物资点"
    │
    ├─ 队员走到物资点
    ├─ 使用默认搜刮模式（安静搜刮）← 不给玩家添乱
    ├─ 玩家可以点击搜刮进度条 → 切换搜刮模式（安静/快速/暴力）
    └─ 搜刮完成 → 物资自动入队（共享背包）
        └─ 回到跟随状态
```

### 5.2 不会发生的自主行为

```
AI 队员在搜刮场景中：
  ✗ 不会主动搜刮物资（即使感知到隐藏物资点，只提醒不行动）
  ✗ 不会主动进入未探索房间（可能触发更多敌人）
  ✗ 不会在战斗中进行搜刮
  ✗ 不会在敌人靠近时搜刮（警戒优先）
```

## 六、共享背包规则

### 6.1 背包合并

```
搜刮场景中，队伍的背包是逻辑合并的：

  总容量 = 玩家背包 + AI队员1背包 + AI队员2背包
  物资在逻辑上共享

  但在分配时：
  · 轻物品（绷带、罐头）→ 先装玩家背包（便于丢弃决策）
  · 重物品（木材、金属）→ 先装力量高的队员背包
  · 装备→玩家分配

  撤退时丢弃物资：
  · 紧急撤退：随机丢弃 30% 物资（同比例从各背包抽取）
  · 如果某队员战死 → 其背包物资掉落在尸体位置（可捡回）
```

### 6.2 负重与移动

```
负重影响移动速度：

  背包容量使用率 < 50%：正常速度
  背包容量使用率 50-80%：移速 -10%
  背包容量使用率 80-100%：移速 -20%

队伍移动速度 = min(队长移速, 队员1移速, 队员2移速)
  → 如果某个队员负重过大，拖慢整个队伍
  → 策略层面：是否让该队员减负？
```

## 七、撤退中的 AI 行为

### 7.1 正常撤退

```
玩家点击"返回基地"后：

  所有队员中断当前动作（搜刮/战斗）
  → 向撤离点移动（自动寻路）
  → 如果有敌人追击：
      · 血量 > 50% 的队员自动殿后（攻击追击敌人，边打边退）
      · 血量 < 50% 的队员优先跑向撤离点
  → 所有队员到达撤离点 → 撤退成功
  → 如果有队员在撤退中倒地 → 弹出抉择：
      · 回去救（风险：全队可能被拖住）
      · 放弃（永久失去该队员，士气 -30）
```

### 7.2 暴风雪滞留

```
如果倒计时归零未撤离：

  所有队员开始承受暴风雪伤害（独立检定）
  → AI 队员自主行为：
      · 寻找最近的庇护点（破屋、车辆残骸、洞穴）
      · 如果找到了 → 躲进去等暴风雪结束
      · 如果没有庇护 → 每 10 秒检定，冻伤累积
  → 暴风雪结束后：
      · 存活队员 → 场景切换为"暴风雪后"状态（敌人位置重置，可能有新尸体）
      · 玩家仍可继续搜刮或立即撤退
```

## 八、队员特殊能力在搜刮中的应用

每个幸存者的特性和装备在搜刮中有具体体现：

| 特性/装备 | 在搜刮中的效果 |
|-----------|--------------|
| 高感知 | 发现隐藏物资点的概率 ×2；陷阱可见 |
| 高力量 | 可单人暴力破拆（无需撬棍）；携带重物不减移速 |
| 高耐力 | 暴风雪暴露伤害 -30% |
| 装备·开锁器 | 可打开上锁柜而不触发噪音 |
| 装备·探测器 | 发现电子设备中的隐藏数据 |
| 装备·医疗包 | 战斗中可治疗队友；倒地时可自救 |
| 特质·勇敢 | 撤退时殿后概率 +50% |
| 特质·胆小 | 血量 < 60% 时拒绝执行危险指令（需玩家二次确认） |

## 九、与基地 AI 的切换

```
场景切换时的状态保存与恢复：

从基地 → 搜刮场景：
  · 队员暂停基地 AI 状态机
  · 保存当前任务进度（如"建造墙壁 67%"）
  · 切换到搜刮 AI 模式

从搜刮场景 → 基地：
  · 队员恢复基地 AI 状态机
  · 恢复之前暂停的任务
  · 搜刮中的伤/冻伤带入基地
  · 搜刮中的士气变化 + 收获 → 更新基地状态

如果队员在搜刮中死亡：
  → 基地中该幸存者被移除
  → 触发同伴死亡事件（士气 -30）
```

---

> 搜刮中的 AI 队友设计核心：**自主但不越位**。战斗时自己打，不需要微操；搜刮时等指令，不乱动。玩家感觉到的是"有人在帮我看后背"，而不是"又多了两个需要照顾的宠物"。
