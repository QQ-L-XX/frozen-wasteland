## 一、温度与热力系统

### 1.1 基础公式

```
房间温度(下一帧) = 房间温度(当前) + Δt × ( 供热 + 邻室传导 - 热损 )

其中：
  Δt = 帧间隔（秒）
  供热 = 锅炉输出 × 管道效率 / 房间体积
  管道效率 = 1 / (1 + 管道长度 × 0.02)
  房间体积 = 房间格数（每格计为 1 立方米）

  热损 = (100% - 保温率(%)) × (室内温度 - 室外温度) × 0.005
  邻室传导 = Σ(邻室温度 - 本室温度) × 0.01  [通过墙/门传导]
```

### 1.2 温度区间效果表

| 温度区间 | 名称 | 工作效率 | 士气变化 | 冻伤累积 | 物资腐坏 | 水管 |
|----------|------|---------|---------|---------|---------|------|
| ≥ +15°C | 舒适 | ×1.2 | +1/小时 | 缓慢恢复 | 正常(×1.0) | ✓ |
| +5 ~ +15°C | 可接受 | ×1.0 | 0 | 无变化 | 正常 | ✓ |
| 0 ~ +5°C | 偏冷 | ×0.9 | -0.5/小时 | +1/小时 | 变慢(×0.7) | ✓ |
| -5 ~ 0°C | 寒冷 | ×0.75 | -1/小时 | +3/小时 | 变慢(×0.5) | 冻结风险 |
| -20 ~ -5°C | 严寒 | ×0.5 | -2/小时 | +8/小时 | 基本停止 | 冻结 |
| -40 ~ -20°C | 极寒 | ×0.2 | -4/小时 | +20/小时 | 冻结(×0.1) | 冻结 |
| < -40°C | 致命 | 0 | -10/小时 | +50/小时 | 完全冻结 | 冻裂 |

## 二、幸存者状态系统

### 2.1 五状态变化规则

#### 营养（0-100，越高越饱）
```
基础消耗：-2 / 游戏小时（静息）
工作消耗：×1.5（轻劳动）/ ×2.0（重劳动）/ ×3.0（极重劳动）
寒冷加成：温度每低于 0°C → 额外消耗 +0.5 / °C / 小时
进食恢复：+15（简餐）/ +30（正餐）/ +50（大餐）
```

#### 体温（0-100，越高越暖）
```
基础变化：
  在供暖房间内：+房间温度效果（如 +15°C 房 → +5/小时）
  室外：-室外温度效果（如 -20°C → -10/小时）
  供暖装备加成：暖贴 → 室外变化 ×0.7；电热背心 → 室外变化 ×0.4
```

#### 健康（0-100，100=满血）
```
伤害：
  冻饿者攻击：-8~12
  变异兽攻击：-15~25
  掠夺者枪击：-20~35
  BOSS 攻击：-30~50
恢复：
  在居住舱休息：+3/小时（无治疗）
  在医务室+有医生：+10/小时
  使用医疗包：+30（一次性）
```

#### 士气（0-100，越高越好）
```
基础：-1/天（末日被动衰减）
加成：
  舒适温度：+1/小时
  私人房间：+2/天
  好友同在：+1/小时
  吃大餐：+10
  完成任务：+5
惩罚：
  寒冷：-2~10/小时（取决于程度）
  饥饿：-5/小时
  同伴死亡：-30（一次性）
  看到尸体：-5
```

#### 疲劳（0-100，越高越累）
```
累积：+5/小时（醒着）
极重劳动加成：+3/小时
恢复：-15/小时（睡觉）
居住舱舒适度加成：-2/小时
失眠惩罚：恢复 ×0.5
```

### 2.2 冻伤系统

冻伤是独立于体温的累积值（0-100）：

```
冻伤累积 = Σ(冻伤速率 × 时间)

冻伤速率（每游戏小时）：
  温度区间         速率
  ≥ 0°C            0（不累积，缓慢恢复）
  -5 ~ 0°C         +1
  -20 ~ -5°C       +3
  -40 ~ -20°C      +8
  < -40°C          +20

冻伤等级：
  0-20：轻度 → 无惩罚
  20-50：中度 → 工作效率 ×0.8，移速 -10%
  50-80：重度 → 工作效率 ×0.5，移速 -30%，可能冻掉手指/脚趾
  80-100：截肢级 → 工作效率 ×0.2，移速 -50%，需要手术

恢复：
  在温暖环境（≥ +10°C）：-2/小时
  使用冻伤膏：-15（一次性）
  医务室治疗：-5/小时
```

### 3.1 食物消耗

| 幸存者状态 | 日消耗（单位） |
|-----------|--------------|
| 基本 | 2 单位/天 |
| 寒冷环境（<0°C） | 3 单位/天 |
| 重体力劳动 | 3 单位/天 |
| 寒冷+重体力 | 4 单位/天 |
| 伤病恢复中 | 2.5 单位/天 |

### 3.2 食物腐坏

```
腐坏速率 = 基础速率 × 温度系数

基础速率（每天损失的新鲜度 %）：
  罐头：2%
  干粮：5%
  冻肉：15%（常温）；3%（冷冻）
  新鲜蔬菜：25%
  熟食：20%

温度系数：
  ≥ +10°C ：×2.0
  0 ~ 10°C ：×1.0
  -10 ~ 0°C：×0.5
  < -10°C  ：×0.1（几乎不坏）

新鲜度 < 50%：食用效果减半
新鲜度 = 0%：不可食用
```

```
建造时间(秒) = 基础工时 / (建造者力量 × 效率系数)

基础工时：
  木墙/格：20
  加固墙/格：40
  地下掩体墙/格：80
  门：30
  功能设施（锅炉等）：120~600

  单人建造：1.0
  双人协作：1.6（不是 2.0，有协作损耗）
```

## 八、事件触发概率模型

### 8.1 天气事件

由 Perlin Noise 驱动，不设定固定概率，而是生成天气曲线：

```
天气曲线 = PerlinNoise(时间种子)
  → 映射到温度/风速/降雪量

暴风雪触发条件：
  风速 > 阈值 AND 降雪量 > 阈值
  → 暴风雪概率 = f(风速, 降雪量)
```

### 8.2 基地事件（每日概率）

| 事件 | 基础概率 | 修正 |
|------|---------|------|
| 管道破裂 | 5% | 温度 < -20°C → 15%；维修技能高于队伍中位值 → -3% |
| 火灾 | 2% | 使用火盆 → +5%；有烟囱 → -1% |
| 幸存者冲突 | 3% | 有仇敌关系 → +20%；士气整体低 → +10% |
| 物资被盗 | 1% | 防御设施 < 2 → +8%；有围墙 → -3% |
| 掠夺者袭击 | 3% | 无线电开机 → +5%；积累了 > 100 格物资 → +10% |

## 九、平衡参考线

### 9.1 新手期数值（第 1-5 天）

```
起始资源：
  幸存者：3 人（老兵、技师、医生）
  物资：食物 30 单位、水 20 单位、木材 50、金属 20、煤 20
  建筑：木棚一间（15格，火盆供暖）
  燃料存量：可供暖约 2.5 天

日均消耗：
  食物：3人 × 3单位 = 9 单位/天
  燃料：火盆 × 24小时 = 木材 24 / 天
  → 必须每 2 天外出搜刮一次才能维生
```

### 9.2 自给自足临界点（目标设计）

```
关键节点：
  Day 7-10：建成温室 → 食物自给
  Day 15-20：建成锅炉系统 → 燃料效率翻倍
  Day 30+：地热 → 燃料完全自给

设计意图：
  前 10 天是最紧张的生存期（高强度搜刮 + 饥饿压力）
  10-20 天转入扩建期（需要稀有物资 → 更远的探索）
  20 天以后进入战略期（管理 + 扩展 + 故事）
```
