# 狩猎与采集系统

> 填补 P1 缺失项 | 2026-05-18 | v0.1

---

## 一、定位：为什么要狩猎

搜刮是从废墟中消耗存量物资（有限，枯竭度上升）。狩猎是从荒野中获取可再生资源。

| 维度 | 搜刮 | 狩猎 |
|------|------|------|
| 资源性质 | 存量（会枯竭） | 可再生 |
| 风险 | 敌人 + 暴风雪 | 猎物反击 + 天气 |
| 产出 | 工业品、药品、建材 | 食物、皮毛、骨头 |
| 节奏 | 快（15分钟内决策） | 中速（半自主，派人出去等结果） |

---

## 二、核心方案：半自动化狩猎

搜刮场景已经有 11 阶段深入交互。狩猎不需要再造一个同等深度的独立场景。采用"派人出去，等结果回来"的模式，和搜刮形成节奏互补。

```
狩猎流程：

  1. 在世界地图选择荒野区域
  2. 选择猎人和装备
  3. 派出 → 消耗燃料 + 停留天数
  4. 猎人自动执行（后台检定，不进入场景）
  5. 到时间或手动召回 → 狩猎总结
```

## 四、选择猎人与装备

### 猎人属性影响
- 感知 → 发现猎物概率
- 力量 → 可携带猎物上限
- 耐力 → 荒野停留天数上限

### 装备

| 装备 | 效果 | 风险 |
|------|------|------|
| 徒手+陷阱 | 仅被动捕猎，产出×0.5 | 安全 |
| 弓箭 | 远程安静，产出×1.0 | 低 |
| 猎枪 | 高伤，产出×1.5 | 噪音吸引变异兽 +30% |
| 陷阱工具 | +1 小猎物/天/件 | 无 |
| 诱饵 | 稀有猎物概率 +20% | 可能吸引危险猎物 |

### 停留天数
- 1天：低产出，燃料消耗 1
- 3天：中产出，燃料消耗 3
- 5天：高产出，燃料消耗 5，可能发现稀有猎物

## 五、后台狩猎检定（每日一次）

```
每日狩猎检定（系统自动，不弹窗）：

  掷骰：是否发现猎物？
    ├─ 是 → 什么猎物？
    │       └─ 攻击检定（感知+武器 vs 猎物回避）
    │           ├─ 成功 → 获得猎物
    │           └─ 失败 → 猎物逃走
    │
    ├─ 特殊事件（10%/天）：
    │   ├─ 发现稀有猎物 → 弹出抉择弹窗
    │   ├─ 遇到其他猎人 → 社交或战斗
    │   └─ 发现隐藏地点 → 标记在地图上
    │
    └─ 风险检定：
        ├─ 天气恶劣 → 受伤/冻伤
        ├─ 猎物反伤 → 血量降低
        └─ 迷失方向 → 返回延迟 1 天
```

## 八、猎物处理

带回基地后自动处理（如果有空闲猎人/农民）：
- 皮毛 → 仓库（建材/交易品）
- 肉 → 仓库（食物，会腐坏）
- 角/牙/爪 → 仓库（稀有材料，制造高级装备）

处理时间：小猎物 10 秒 / 中猎物 30 秒 / 大猎物 60 秒。

## 九、与工作优先级联动

```
狩猎在优先级表中的激活条件：
  1. 世界地图有丰度 > 20% 的荒野区域
  2. 猎人有武器（弓箭或猎枪）
  3. 基地食物存量 < 5 天（自动触发）；或玩家手动调高优先级

条件不满足时，猎人转而执行其次优先级任务。
```
