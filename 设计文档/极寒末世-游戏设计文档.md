## 一、游戏概述

### 1.1 一句话描述
在永不终结的核冬天里，带领幸存者搜刮废墟、建设供暖基地、在冰封世界中重建文明。

### 1.2 核心体验
- **温度即生命**：极寒不只是背景板，而是贯穿所有系统的生存压力。
- **搜刮即冒险**：每次出门都是一次赌博——拿得越多越危险，走晚了就回不来。
- **基地即家园**：每一堵墙、每一根管道都是亲手建造，供暖网络连接的不只是房间，还有人心。

### 1.3 核心玩法循环

```
外出探索 → 收集物资 → 返回基地 → 建造/升级/供暖 → 下一次更远的探索
    ↑                                                      │
    └────────────────── 极寒倒逼 ──────────────────────────┘
```

### 1.4 参考融合

| 参考游戏 | 取什么 | 不用什么 |
|----------|--------|----------|
| **环世界 (RimWorld)** | 2D俯视角、工作优先级、时间管理、小人故事 | 过于复杂的器官/交易系统 |
| **辐射4 (Fallout 4)** | 模块建造、管线连接、世界地图探索 | 第一人称、琐碎垃圾收集 |
| **僵尸世界大战 (WWZ)** | 搜刮节奏、噪音-敌人反馈、撤退抉择 | 纯射击、线性关卡 |

### 1.5 游戏类型
- 视角：2D 俯视角（可缩放）
- 操作：可暂停实时（RTwP）
- 类型：生存管理 + 基地建设 + 探索搜刮

## 二、世界观

### 2.1 设定
- **核冬天**：全球核冲突后，尘埃遮蔽阳光，地球进入永冬。
- **时间**：灾难后约 5-10 年，城市被冰雪半掩埋，幸存者在地表以下挣扎求生。
- **地点**：一座被冰封的工业城市及其周边废墟带。

### 2.2 视觉基调
- 白色（雪） + 灰色（废墟） + 暖橙色（火光/供暖）为主要色板
- 基地内暖色调灯光与室外蓝白冷色调形成强烈对比
- 暴风雪来临时可见度骤降，世界被白色吞噬

## 四、温度系统（贯穿所有层的核心）

### 4.1 温度作为通用约束

| 层 | 温度的影响 |
|----|-----------|
| 基地 | 房间温度决定：工作效率、健康、物资保质期、士气 |
| 世界地图 | 出行消耗燃料（=热量预算），极端天气封锁区域 |
| 搜刮场景 | 暴露时间上限，供暖装备消耗，冻伤累积 |

### 4.2 温度数值模型（初设）

| 温度区间 | 状态 | 效果 |
|----------|------|------|
| +15°C 以上 | 舒适 | 工作效率 ×1.2，士气缓慢上升 |
| +5 ～ +15°C | 可接受 | 正常 |
| -5 ～ +5°C | 寒冷 | 工作效率 ×0.8，冻伤风险缓慢累积 |
| -20 ～ -5°C | 严寒 | 工作效率 ×0.5，冻伤快速累积，水管冻结 |
| -40 ～ -20°C | 极寒 | 无法室外工作，裸露物资开始损坏 |
| -40°C 以下 | 致命 | 几分钟内死亡 |

### 4.3 基地热力模拟（简化版）

```
房间温度变化 = 锅炉供热 - 建筑热损 - 开门热损 + 相邻房间传导

热损 = (100% - 保温率) × 内外温差系数
管道传输热损 = 管道距离 × 每格损耗
```

- 热量从锅炉房通过管道向各房间扩散
- 可以用**热力图叠加**在基地上显示温度分布
- 漏热点会在墙壁上以冷气流线可视化标记

### 4.4 暴风雪系统
- 由 Perlin Noise 驱动的天气节奏
- 暴风雪前有预警（气压下降、天色变化）
- 暴风雪期间：室外温度骤降 15-25°C、可见度降低、积雪累积
- 暴风雪后：需要铲雪、修补损坏、可能发现被埋的新区域

## 五、物资系统

### 5.1 生存层（不拿就死）

| 类别 | 物品举例 | 特殊机制 |
|------|---------|---------|
| 食物 | 罐头、冻肉、维生素片、干粮 | 保质期 + 腐烂；温度影响腐烂速度 |
| 净水 | 瓶装水、净水片 | 雪可以煮成水但消耗燃料 |
| 燃料 | 煤、木柴、丙烷罐、废塑料 | 不同燃料效率和副作用不同 |
| 药品 | 抗生素、绷带、冻伤膏、止痛药 | 过期后效果降低或有害 |

### 5.2 建造层

| 类别 | 物品举例 | 用途 |
|------|---------|------|
| 基础建材 | 木材、金属、石材 | 墙壁、地板、屋顶 |
| 绝缘材料 | 碎布、泡沫板、玻璃纤维、皮毛 | 提升建筑保温率 |
| 电子元件 | 电路板、电池、电线、马达 | 高级设施和科技 |
| 种植材料 | 玻璃/薄膜、土壤、种子 | 温室建设 |

### 5.3 科技层

| 类别 | 物品举例 | 解锁方式 |
|------|---------|---------|
| 蓝图 | 建筑图纸、设备设计图 | 搜刮获得，在工坊解读 |
| 知识 | 书籍、硬盘、技术手册 | 指派幸存者学习 |
| 稀有零件 | 发电机核心、芯片、精密轴承 | 仅工业区/军事区产出 |

### 5.4 交易层
- **硬通货**：子弹（最稳定）、药品、伏特加、电池
- 与其他幸存者聚落的贸易需要无线电联络后开启

## 六、基地建设

### 6.1 建造方式
- **模块化放置**：墙壁、地板、屋顶、门、窗各自独立放置
- **功能设施**：锅炉、温室外壳、工坊工作台、医务床、无线电塔、防御炮塔
- **管道铺设**：连接锅炉与各房间的供暖管线（类似辐射4的电路逻辑）
- 放置时实时显示保温值变化

### 6.2 建筑等级

| 等级 | 材料 | 保温率 | 耐久 | 建造时间 |
|------|------|--------|------|----------|
| 木棚 | 木材 | 40% | 低 | 短 |
| 木屋 | 木材+绝缘 | 60% | 中 | 中 |
| 加固屋 | 木材+金属+绝缘 | 75% | 高 | 长 |
| 地下掩体 | 金属+水泥+绝缘 | 90% | 极高 | 极长 |

### 6.3 供暖技术层级

| 层级 | 热源 | 供热范围 | 燃料效率 | 解锁条件 |
|------|------|---------|---------|---------|
| T0 | 火盆 | 单房间 | 极低 | 初始 |
| T1 | 煤炉 | 单房间 | 低 | 搜刮煤+金属 |
| T2 | 锅炉+管道 | 多房间(管道网络) | 中 | 搜刮锅炉蓝图 |
| T3 | 高效锅炉 | 全基地 | 高 | 工坊研发 |
| T4 | 地热井 | 全基地 | 极高(免费) | 稀有蓝图+深层挖掘 |
| T5 | 核能供热 | 全基地+区域供热 | 极高 | 军事区核心蓝图 |

### 6.4 功能房间

| 房间 | 必需温度 | 功能 | 升级路径 |
|------|---------|------|----------|
| 居住舱 | ≥ +5°C | 睡觉、恢复 | → 私人房间（士气+） |
| 锅炉房 | 无要求 | 产热 | → 更高级热源 |
| 温室 | ≥ +2°C | 种植食物 | → 照明升级→速成作物 |
| 工坊 | ≥ -5°C | 制造/维修/解读蓝图 | → 精密工坊→解锁高级科技 |
| 仓库 | ≥ -20°C | 储存物资 | → 冷藏库/恒温库 |
| 医务室 | ≥ +10°C | 治疗伤/病 | → 手术室→义肢 |
| 无线电室 | ≥ 0°C | 发现新区域/联络聚落 | → 信号增强→跨区域通讯 |
| 防御设施 | 无要求 | 防御掠夺者袭击 | → 自动炮塔 |

### 6.5 建设中的策略抉择
- **空间 vs 保温**：大空间需要更多供暖，紧凑基地更省燃料
- **地上 vs 地下**：地下保温好但挖掘成本极高；地上建造快但热损大
- **集中 vs 分散**：一个锅炉供所有房间效率高，但管道越长热损越大
- **防御 vs 扩展**：围墙和炮塔消耗材料但抵御袭击；快速扩张但暴露面大

## 七、幸存者系统

### 7.1 属性

| 属性 | 范围 | 影响 |
|------|------|------|
| 力量 | 1-10 | 建造速度、携带重量 |
| 智力 | 1-10 | 科技解读速度、医疗效果 |
| 耐力 | 1-10 | 抗寒能力、连续工作时间 |
| 感知 | 1-10 | 搜刮时发现隐藏物资、预警敌人 |

### 7.2 工作优先级

| 幸存者 | 建造 | 狩猎 | 种植 | 医疗 | 维修 | 搜刮 | 搬运 |
|--------|------|------|------|------|------|------|------|
| 老兵 | 3 | 4 | / | 2 | 2 | 1 | 3 |
| 技师 | 1 | / | / | / | 1 | 3 | 4 |
| 医生 | / | / | 2 | 1 | / | / | 3 |
| 农民 | 4 | / | 1 | / | 3 | 4 | 2 |

### 7.3 状态系统

| 状态条 | 机制 |
|--------|------|
| 健康 | 受伤/生病 → 需要医务室治疗 |
| 营养 | 饥饿 → 工作效率下降 → 死亡 |
| 体温 | 低温 → 冻伤累积 → 截肢/死亡 |
| 士气 | 低 → 崩溃风险（怠工/暴力/叛逃） |
| 疲劳 | 缺觉 → 效率下降 → 昏倒 |

### 7.4 性格与故事
- 每个幸存者有**背景故事**和**性格特质**
- 特质影响行为（悲观→低温下士气降更快；自私→可能偷藏物资）
- 幸存者之间会产生**关系**（友好/敌对）
- **道德抉择**贯穿始终

## 十一、叙事与故事生成

### 11.1 叙事哲学
- **不写死剧本**：没有强制主线。故事由系统交互中涌现（环世界式的 emergent storytelling）。
- **场景即叙事**：每个废墟本身就是一段沉默的故事——一张全家福、一封未寄出的信、一个儿童的玩具。
- **抉择即角色**：你是谁，由你在道德岔路口的选择定义。

### 11.2 故事种子系统

世界预设了若干 **"故事种子"**，它们不是任务，而是等待被发现的线索：

| 种子类型 | 举例 |
|----------|------|
| 地点种子 | "医院地下室的隔离病房被从外面焊死"——里面的东西还活着吗？ |
| 人物种子 | 无线电里一个孩子的声音，每 3 天在同一频率出现一次 |
| 物品种子 | 一本日记，记录了一个家庭的最后 30 天——最后一页在第 23 天 |
| 组织种子 | 一支武装车队定期经过东区——他们是军队残余，还是新的暴君？ |

玩家是否追踪这些种子完全自主。追踪产生支线叙事，不追踪它们也会在背景中自行演变。

### 11.3 幸存者个人叙事
- 每个幸存者有一个**个人问题**待解决（寻找失散的家人、治愈旧伤、完成未竟的事业）
- 解决这些问题提升士气并解锁特殊能力
- 这些问题与世界观中的故事种子可能交叉

## 十二、UI/UX 设计方向

### 12.1 设计原则
- **冷暖对比是核心视觉语言**：UI 色调随温度变化——基地温暖时 UI 偏暖橙，寒冷区域 UI 偏蓝白
- **信息分层**：第一层只显示关键生存指标（温度、食物、燃料天数）；第二层展开具体数据
- **极简主义**：末日不需要华丽界面——功能优先，装饰最小化

### 12.2 关键 UI 面板

| 面板 | 功能 | 视觉方向 |
|------|------|---------|
| 主 HUD | 温度条、日期/时间、幸存者数、警报 | 顶部窄条，冰蓝色调 |
| 基地总览 | 房间列表 + 热力图叠加 | 半透明面板覆盖在游戏画面 |
| 工作优先级 | 幸存者×工作任务矩阵 | 表格+颜色编码 |
| 世界地图 | 区域选择 + 风险/距离标注 | 手绘风格地图，雪痕纹理 |
| 搜刮界面 | 背包格子 + 噪音条 + 时间倒计时 | 高对比度，紧迫感 |
| 事件弹窗 | 道德抉择 / 幸存者事件 | 文字为主，选项清晰 |

### 12.3 操作流程
- **基地模式**：点击房间→展开管理面板；点击幸存者→指派/检查状态
- **建造模式**：从底部建造栏拖拽模块到网格；管道以连线方式铺设
- **世界地图**：点击区域→显示信息→"派出探索队"按钮
- **搜刮模式**：点击物资点→选择搜刮方式→实时移动+战斗

## 十三、音效与氛围设计

### 13.1 音景层次
| 层 | 内容 | 作用 |
|----|------|------|
| 环境底噪 | 风啸、雪落、建筑吱嘎 | 持续的空间感 |
| 基地音效 | 锅炉嗡鸣、人声、建造敲击 | 基地"活着"的感觉 |
| 天气 | 暴风雪渐进增强、雪后寂静 | 节奏信号 |
| 搜刮 | 脚步声、开箱、敌人低吼 | 紧张感 |
| 音乐 | 仅在关键时刻出现 | 不让音乐冲淡孤独感 |

### 13.2 关键音效设计
- **温度警报音**：温度跌破阈值时，低沉的金属嗡鸣渐入
- **暴风雪逼近**：风声从远处呼啸逐渐逼近，高频杂音增加
- **噪音反馈**：搜刮时每个动作都有清晰的音效层级
