> 填补 GDD 缺失项 | 2026-05-18 | v0.1

## 一、通用 AI 架构

所有敌人共享一个三层状态机：

```
【层1】默认行为 —— 巡逻 / 闲置 / 睡眠
【层2】感知响应 —— 听到噪音 / 看到玩家 / 闻到血腥
【层3】战斗行为 —— 追击 / 攻击 / 包围 / 撤退
```

## 二、冻饿者 —— 基础敌人

### 2.1 行为特征
- 曾经是平民，被极寒逼疯
- 视觉差（近视，索敌范围 8 格），但**听觉灵敏**
- 行为模式：缓慢巡逻 → 听到噪音靠近 → 看到玩家后变成暴走模式

### 2.2 状态机

```
                ┌──────────┐
    游戏开始 → │  巡逻    │ ← 沿预设路线缓慢走动
                └────┬─────┘
                     │
         ┌───────────┼───────────┐
         ▼           ▼           ▼
    听到噪音      嗅到玩家     看到玩家
    (范围15格)   (范围5格)    (范围8格)
         │           │           │
         ▼           ▼           ▼
    ┌─────────┐ ┌─────────┐ ┌─────────┐
    │ 警戒靠近 │ │ 嗅探追踪 │ │ 暴走追击 │
    │ 走向声源 │ │ 走向气味 │ │ 直线冲刺 │
    └────┬────┘ └────┬────┘ └────┬────┘
         │           │           │
         └───────────┼───────────┘
                     ▼
              看到玩家在攻击范围？
                │         │
               是        否
                │         │
                ▼         ▼
          ┌────────┐  ┌──────┐
          │  攻击  │  │ 追击 │
          └───┬────┘  └──┬───┘
              │          │
              ▼          ▼
          玩家离开    丢失目标 > 5 秒
          攻击范围？     │
          是│否         ▼
            │      ┌──────────┐
            │      │ 回到巡逻 │
            │      └──────────┘
            │
            ▼
         继续攻击
```

### 2.3 巡逻逻辑

```
巡逻路线 = 预设路径点（由场景生成时放置） + 随机偏移

每个冻饿者有 3-5 个路径点：
  · 站岗点（停留 5-15 秒，左右张望）
  · 巡逻路径（走最近邻路）
  · 随机闲逛半径（±3 格，在路径点上随机偏移）

多个冻饿者之间：
  · 不会重叠（简单的避开碰撞）
  · 听到噪音时，最近的 1-2 个响应，不是全体出动
```

### 2.4 暴走模式

```
触发：冻饿者看到玩家（进入 8 格视野）

行为变化：
  · 移速 ×2（慢→快）
  · 攻击频率 ×1.5
  · 不再巡逻——持续追击直到丢失目标或死亡
  · 发出嘶吼（噪音 +10，可能吸引附近其他冻饿者）

视觉反馈：
  · 头顶出现红色感叹号
  · 行走动画变为冲刺动画
  · 屏幕边缘微微泛红（如果玩家被追击）
```

### 2.5 冻饿者属性

| 参数 | 值 |
|------|-----|
| 血量 | 30 |
| 攻击力 | 5-10 |
| 基础移速 | 2 格/秒 |
| 暴走移速 | 4 格/秒 |
| 视野范围 | 8 格（锥形，前方 120°） |
| 听觉范围 | 15 格（全向） |
| 嗅觉范围 | 5 格（全向，仅对受伤流血的玩家） |
| 巡逻速度 | 1 格/秒 |

## 三、变异狼 —— 群体猎手

### 3.1 行为特征
- 辐射 + 极寒产生的变异狼
- 群体狩猎（2-5 只一组）
- 会**包抄**——从不同方向逼近
- 有"头狼"概念——头狼死亡后剩余狼可能撤退

### 3.2 包抄算法

```
假设狼群 = {头狼, 左翼, 右翼, 后备}

遭遇玩家时：

头狼：
  · 正面逼近（最短路径），吸引注意力
  · 接近到 5 格时转为横向移动，不给玩家稳定射击角度

左翼：
  · 向玩家左侧绕行（偏移 30°-45°）
  · 目标：到达玩家侧后方 3-5 格处
  · 路径计算：目标点 = 玩家位置 + rotate(玩家朝向, -45°) × 6格

右翼：
  · 同左翼但偏移 +45°

后备狼：
  · 保持在头狼后方 5-8 格
  · 如果左翼或右翼被杀 → 后备狼补位

攻击时机：
  · 当至少 2 个方向的狼都进入攻击范围（3格）→ 同时发动攻击
  · 否则等待包抄就位
```

### 3.3 头狼机制

```
头狼：
  · 血量 ×1.5（75 vs 普通 50）
  · 不会主动进入玩家近战范围（保持 3-5 格）
  · 头狼死亡 → 剩余狼进行"士气检定"

士气检定（头狼死后，每只剩余狼独立掷骰）：
  剩余狼 ≥ 2：70% 概率撤退
  剩余狼 = 1：90% 概率撤退
  撤退行为：跑向地图边缘 → 消失

如果撤退检定全部失败 → 剩余狼进入狂暴模式（攻击×1.5，不再包抄直接冲）
```

### 3.4 变异狼属性

| 参数 | 普通 | 头狼 |
|------|------|------|
| 血量 | 50 | 75 |
| 攻击力 | 10-18 | 15-22 |
| 移速 | 5 格/秒 | 5 格/秒 |
| 视野 | 12 格 | 15 格 |
| 嗅觉 | 10 格 | 15 格 |
| 群规模 | 3-5 | — |
| 掉落 | 皮毛×2 | 皮毛×3 + 狼牙(稀有) |

## 四、变异熊 —— 独行坦克

### 4.1 行为特征
- 巨兽级单体
- 守护巢穴（工业区深处）
- 不主动巡逻，但感知范围极大
- 攻击有**击退效果**（玩家被拍飞 3 格）

### 4.2 行为状态

```
        ┌──────────┐
        │   睡眠   │ ← 初始状态，玩家接近到 20 格开始苏醒
        └────┬─────┘
             │ 玩家进入 20 格范围
             ▼
        ┌──────────┐
        │   苏醒   │ ← 站立咆哮动画（3 秒），此期间无敌+不攻击
        └────┬─────┘
             │
             ▼
        ┌──────────┐
        │   追击   │ ← 直线冲向玩家，不绕路
        │          │   路过障碍物 → 撞碎（木材/碎石可摧毁）
        └────┬─────┘
             │ 进入攻击范围(2格)
             ▼
        ┌──────────┐
        │   攻击   │ ← 3 种攻击模式随机/条件触发：
        │          │   ① 挥击：正面 2×3 格，击退 3 格
        │          │   ② 冲撞：直线冲刺 5 格，撞到墙对自己造成反伤
        │          │   ③ 震地：自身 3 格半径 AOE，伤害低但减速
        └────┬─────┘
             │ 玩家跑出 25 格？
             ▼
        ┌──────────┐
        │ 返回巢穴 │ ← 缓慢走回初始位置，不回血
        └──────────┘
```

### 4.3 变异熊属性

| 参数 | 值 |
|------|-----|

| 移速 | 3 格/秒（中速） |

## 五、掠夺者 —— 有组织的人类敌人

### 5.1 行为特征
- 其他幸存者群体，不疯但充满敌意
- 会**设陷阱**和**利用掩体**
- 有基本战术：火力压制 + 侧翼包抄
- 会**捡尸体上的武器**（动态武装）

### 5.2 陷阱类型

| 陷阱 | 触发 | 效果 | 玩家应对 |
|------|------|------|---------|
| 绊线 | 走过触发线（隐形） | 摔倒 2 秒 + 噪音 +15 | 高感知队员可见 |
| 捕熊夹 | 踩到触发格 | 伤害 15 + 定身 5 秒 | 可用撬棍解除 |
| 警示铃 | 走过触发线 | 噪音 +30，全体掠夺者进入战斗状态 | 高感知队员可见 |
| 燃烧瓶 | 掠夺者投掷 | 落点 2×2 格着火，持续 20 秒 | 移开 |

### 5.3 战术状态机

```
                ┌──────────┐
    初始 →     │  巡逻/站岗 │
                └────┬─────┘
                     │ 发现玩家
                     ▼
                ┌──────────┐
                │  警报！   │ ← 1 个掠夺者吹哨/喊叫
                │ 全员战斗  │   附近的掠夺者全部切换战斗状态
                └────┬─────┘
                     │
         ┌───────────┼───────────┐
         ▼           ▼           ▼
    ┌────────┐ ┌────────┐ ┌────────┐
    │ 火力手 │ │ 突击兵 │ │ 投弹兵 │
    │ 找掩体 │ │ 侧翼绕 │ │ 找角度 │
    │ 远程射 │ │ 近战冲 │ │ 扔燃烧 │
    └────┬───┘ └────┬───┘ └────┬───┘
         │          │          │
         └──────────┼──────────┘
                    ▼
            火力手伤亡 > 50%？
             │         │
            是        否
             │         │
             ▼         ▼
        ┌────────┐  继续战斗
        │ 全员撤退│
        └────────┘
```

### 5.4 掩体利用

```
掠夺者移动时会优先走到掩体后方（障碍物/墙壁/车辆残骸）。
掩体提供：
  · 玩家命中率 -30%（如果掠夺者在掩体后）
  · 掠夺者可探头射击（每 3-5 秒探头 1 次，每次暴露 0.5 秒）

掩体优先级：
  墙壁转角 > 车辆残骸 > 柱子 > 开放地面
```

### 5.5 掠夺者属性

| 参数 | 火力手 | 突击兵 | 投弹兵 |
|------|--------|--------|--------|
| 血量 | 60 | 70 | 50 |
| 武器 | 手枪/步枪 | 撬棍/小刀 | 手枪+燃烧瓶×2 |
| 攻击力 | 15-25 | 8-20 | 弹12-18 / 火20 |
| 射程 | 12 格 | 1 格（近战） | 8 格（投掷） |
| 移速 | 3 格/秒 | 4 格/秒 | 3 格/秒 |
| 掉落 | 武器+弹药 | 武器+弹药 | 武器+燃烧瓶配方 |

## 六、冰原巨兽 —— BOSS 级敌人

### 6.1 设计目标
- 不是高血量的木桩，而是一个**有阶段的解谜战斗**
- 每个阶段改变行为和弱点
- 研究中心等高级区域限定（每局游戏只有 1-2 个 BOSS）

### 6.2 战斗阶段

```
【阶段 1：完整形态 · 血量 100-70%】

外观：覆盖着厚厚冰甲的巨兽

行为：
  · 缓慢移动，直线追击
  · 攻击模式：
    A. 踩踏（正面 3×3，击退+伤害）
    B. 冰锥喷射（直线 8 格，可躲）

弱点：无。此阶段攻击伤害仅造成 25%。

策略提示：
  → 玩家需要先破坏冰甲
  → 使用热源武器（燃烧瓶、火盆废料）→ 对冰甲造成 3× 伤害
  → 或集中攻击同一部位 → 冰甲局部碎裂

────────────────────────────────────

【阶段 2：冰甲破碎 · 血量 70-40%】

过渡：巨兽咆哮，冰甲剥落

外观：暴露出暗红色的皮肤

行为：
  · 移动速度 +30%
  · 新增攻击：
    C. 狂暴冲撞（直线冲刺 8 格，撞墙自己受伤 5%）
    D. 震地波（全屏，伤害低但必定命中）

弱点：暴露的皮肤。所有攻击恢复正常伤害（100%）。

────────────────────────────────────

【阶段 3：濒死咆哮 · 血量 40-0%】

过渡：巨兽进入狂暴状态

行为：
  · 攻击频率 ×2
  · 所有攻击伤害 ×1.5
  · 无视掩体，能撞碎一切可破坏的墙壁和障碍
  · 移动速度 ×1.5

弱点：背部。巨兽冲刺后会短暂转身（1-2 秒窗口期），
      此时攻击背部造成 2× 伤害，并使其硬直。

环境利用：
  → 引诱巨兽冲向冰裂 → 巨兽掉入冰裂 → 秒杀（如果场景有冰裂）
  → 引诱巨兽冲向支撑柱 → 建筑坍塌 → 巨兽被埋
```

### 6.3 冰原巨兽属性

| 参数 | 值 |
|------|-----|
| 血量 | 300 |
| 冰甲（阶段1减伤） | 75% |
| 攻击力 | 30-50 |
| 移速 | 2 格/秒（阶段1）→ 3（阶段2）→ 4.5（阶段3） |
| 感知范围 | 全图（一旦进入 BOSS 房间就激活） |
| 掉落 | 稀有蓝图×1 + 巨兽皮毛×10 + 巨兽核心(唯一) |

## 七、通用 AI 工具函数（伪代码）

```python
def perceive(world_state, enemy):
    """敌人感知检测"""
    heard = False
    spotted = False
    
    # 听觉检测
    if world_state.noise_level > 0:
        distance_to_noise = dist(enemy.pos, world_state.noise_epicenter)
        if distance_to_noise < enemy.hearing_range:
            heard = True
    
    # 视觉检测（锥形视野）
    if distance_to_player < enemy.sight_range:
        angle = angle_between(enemy.facing, player.pos - enemy.pos)
        if abs(angle) < enemy.fov / 2:
            if not blocked_by_wall(enemy.pos, player.pos, world_state):
                spotted = True
    
    return PerceptionResult(heard, spotted)

def choose_action(perception, enemy_state, world_state):
    """根据感知和状态选择行为"""
    if perception.spotted:
        return Action.CHASE_AND_ATTACK
    elif perception.heard:
        return Action.INVESTIGATE_NOISE
    else:
        return Action.PATROL

def wolf_flank(wolf, pack, player_pos):
    """狼群包抄逻辑"""
    role = wolf.role  # 'alpha', 'left', 'right', 'reserve'
    
    if role == 'alpha':
        return move_toward(player_pos, maintain_distance=5)
    elif role == 'left':
        flank_pos = player_pos + rotate(Vector(6,0), -40°)
        return move_toward(flank_pos)
    elif role == 'right':
        flank_pos = player_pos + rotate(Vector(6,0), +40°)
        return move_toward(flank_pos)
    elif role == 'reserve':
        alpha_pos = pack.alpha.pos
        return move_toward(alpha_pos + Vector(0, -6))
```

> 敌人 AI 的核心原则：**每种敌人有一种"玩家需要学会应对"的独特行为**。冻饿者教噪音管理，变异狼教位置意识，变异熊教环境利用，掠夺者教掩体战术，冰原巨兽教阶段识别。不是越来越多血，而是越来越需要不同的应对策略。
