# 搜刮场景 · 完整流程设计

> 关联 GDD 第八章 | 2026-05-18 | v0.1

---

## 一、流程总览

```
【B 层：世界地图】
   │
   ├─ 1. 选择目标区域
   ├─ 2. 选择探索队员
   ├─ 3. 分配装备
   ├─ 4. 出发（消耗燃料）→ 场景加载
   │
   ▼
【C 层：局部搜刮场景】
   │
   ├─ 5. 场景加载完成，倒计时开始
   ├─ 6. 自由探索 & 搜刮
   │     ├─ 发现物资点 → 选择搜刮方式
   │     ├─ 噪音累积 → 触发敌人
   │     └─ 发现隐藏线索
   ├─ 7. 暴风雪预警（最后 30% 时间）
   ├─ 8. 撤退决策点
   │     ├─ 立即撤退 → 安全返回
   │     ├─ 继续搜刮 → 风险递增
   │     └─ 倒计时归零 → 暴风雪降临 → 生存检定
   │
   ▼
【返回基地】
   ├─ 9. 撤退过程（可能有追逃）
   ├─ 10. 回到基地 → 物资入库
   └─ 11. 搜刮总结
```

## 二、阶段 1-3：出发前准备

### 2.1 选择目标区域

区域信息卡片（点击后弹出）：

```
┌─────────────────────────────┐
│  🏥 圣玛丽医院                │
│  风险：★★★☆☆（中）           │
│  主要物资：药品、医疗设备       │
│  可能物资：医疗蓝图（稀有）     │
│  距离：1.2 km / 燃料：3 单位   │
│  枯竭度：██░░░░░░ 23%         │
│  暴风雪窗口：约 12 分钟（实时）  │
│  [派出探索队]                 │
└─────────────────────────────┘
```

### 2.2 选择队员

- 多带人 = 更多背包空间 + 战斗力，但基地人手减少
- 每个队员消耗燃料（距离 × 人数）
- 状态不佳的队员有 debuff

### 2.3 分配装备（每人 4 槽）

| 槽位 | 可选 | 功能 |
|------|------|------|
| 武器 | 猎枪/手枪/撬棍/徒手 | 攻击力+范围 |
| 工具 | 撬棍/开锁器/医疗包/探测器 | 特殊搜刮动作 |
| 供暖 | 基础暖贴/高级暖贴/电热背心 | 延长暴露时间 |
| 背包 | 轻型(15格)/标准(20格)/重型(30格)/雪橇(50格,慢) | 容量+移速 |

### 3.3 特殊物资点

| 类型 | 需要 | 机制 |
|------|------|------|
| 上锁柜 | 开锁器 | 否则只能暴力破拆 |
| 电子设备 | 探测器 | 能发现隐藏蓝图 |
| 尸体 | 无 | 可能触发短叙事 |
| 隐藏点 | 感知检定 | 发现隐藏物资 |
| BOSS巢穴 | 战斗 | 击败后获得稀有品 |

## 四、阶段 7：噪音与敌人

### 4.1 噪音累积

| 动作 | 噪音值 |
|------|--------|
| 安静搜刮 | +2 |
| 快速搜刮 | +8 |
| 暴力破拆 | +25 |
| 奔跑 | +1/秒 |
| 射击 | +15/枪 |
| 噪音衰减 | -0.5/秒 |

### 4.2 阈值反应

| 噪音区间 | 敌人行为 |
|----------|---------|
| 0-30% | 正常巡逻 |
| 30-60% | 巡逻向噪音源偏移 |
| 60-85% | 触发敌人潮（3-5个） |
| 85-100% | BOSS苏醒+全图搜索 |

### 4.3 战斗

- 实时可暂停（RTwP）
- 受伤→背包物资可能掉落
- 重伤→需搀扶（两人减速）
- 濒死→必须立即医疗包，否则永久死亡

## 五、阶段 8：暴风雪逼近

| 剩余时间 | 视觉 | 音效 | 效果 |
|----------|------|------|------|
| 100-30% | 正常 | 远处风声 | 无影响 |
| 30-15% | 屏幕边缘结冰 | 风声渐强 | 暴露伤害×1.5 |
| 15-5% | 冰霜蔓延，可见度降 | 风声咆哮 | 暴露伤害×2，移速-20% |
| 5-0% | 大半结冰，偏蓝 | 暴风雪全音量 | 暴露伤害×3，移速-40% |
| 0% | 暴风雪降临 | — | 进入生存检定 |

## 六、阶段 9：撤退决策——核心高潮

### 撤退方式

| 方式 | 条件 | 结果 |
|------|------|------|
| 正常撤退 | 任何时间 | 15秒走到撤离点，安全返回 |
| 紧急撤退 | 任何时间 | 丢下30%物资，5秒撤离 |
| 被迫滞留 | 倒计时归零 | 进入暴风雪生存检定 |

### 设计意图

> "背包还有15格，楼上药房还没搜，暴风雪只剩2分钟。走不走？"

——搜刮场景的核心张力就是这个决策时刻。

### 追逃

噪音>30%撤退时，敌人追击。边打边退，被击倒可能失去队员或物资。

## 七、阶段 10：暴风雪生存检定（倒计时归零）

```
┌──────────────────────────────────────┐
│  ⚠ 暴风雪降临！室外 -52°C              │
│  必须立即找庇护！                      │
│  · 药房（需撬锁，60秒）                │
│  · 地下室（40米外）                    │
│  · 救护车（50%遮风）                   │
│  每10秒一次生存检定（耐力+装备 vs 环境） │
└──────────────────────────────────────┘
```

- 检定失败 → 冻伤累积 → 满即死
- 撑过暴风雪 → 生还但状态极差
- 可故意滞留拼一把高级物资——高风险高回报

## 九、区域类型变体

| 区域 | 场景特点 | 节奏 |
|------|---------|------|
| 室内建筑 | 走廊+房间，门可开关 | 逐步推进 |
| 开放废墟 | 半开放，雪覆盖 | 视野开阔但暴风雪影响大 |
| 工业区 | 大空间+多层 | 垂直探索，需要工具 |
| 地铁网络 | 线性+分支隧道 | 路径选择博弈 |
| 军事区 | 设防+自动炮塔 | 先解防御再搜刮 |
| 荒野 | 开放雪原+洞穴 | 追踪脚印/血迹 |

## 十一、数值参考（初设）

| 参数 | 初设值 |
|------|--------|
| 标准搜刮窗口 | 10-15分钟（实时） |
| 安静搜刮耗时 | 6-10秒/物资点 |
| 快速搜刮耗时 | 2-4秒 |
| 暴力破拆耗时 | 1-2秒 |
| 噪音衰减 | -0.5/秒 |
| 敌人潮规模 | 3-5个（60%阈值） |
| 背包基础容量 | 20格（标准） |
| 暴露伤害 | -1%体温/秒（-20°C时） |

> 搜刮场景是三层架构中最快的一层——10-15分钟内完成探索、搜刮、战斗、撤退的全部决策。节奏控制和信息呈现是核心。
