# 载具系统

> 填补科技树 T3「载具改装」 | 2026-05-18 | v0.1

---

## 一、定位

载具改变的是**世界地图层的规则**——探索半径、燃料效率、携带量。没有载具时，探索队步行往返，距离严格受燃料和暴露时间限制。有了载具，可探索区域翻倍，但带来了新的资源消耗和维护成本。

```
步行时代：         探索半径 0-3 km，最大携带量 = 背包容量
载具时代：         探索半径 0-12 km，最大携带量 = 载具货舱 + 背包
```

载具不进入搜刮场景——停在区域边缘，作为移动撤离点 + 外部仓库。

## 二、四种载具

### 2.1 雪橇（T1 末期 · 无蓝图）

```
需求：木材×8 + 金属×4 + 破布×3 | 工坊建造
速度：步行 ×0.8（人力拖行，拖行者疲劳翻倍）
货舱：30 格
乘员：无（不提供保护）
油耗：无（人力）
维护：每 5 次出行 → 木材×2

作用：搜刮时多 30 格缓冲区，但拖行者在场景中移速 -30%
```

### 2.2 雪地摩托（T3 · 需蓝图）

```
需求：金属×15 + 电子×5 + 马达×1 + 蓝图 | 精密工坊
速度：步行 ×3.0
货舱：20 格
乘员：1-2 人（免暴露伤害）
油耗：燃油 1/km 或 丙烷罐替代
维护：每 3 次出行 → 金属×3 + 电子×1

作用：探索半径扩至 8 km，到达快 → 搜刮窗口相对延长
```

### 2.3 雪地卡车（T3 末期 · 极稀有蓝图）

```
需求：金属×30 + 电子×10 + 马达×2 + 蓝图 | 精密工坊
速度：步行 ×2.0
货舱：100 格
乘员：1 驾驶员 + 3 乘客（全队免暴露伤害）
油耗：燃油 3/km 或 丙烷 1/km
维护：每次出行后 → 金属×5 + 电子×2

作用：移动仓库，可带 4 人队，暴风雪中也能出行
```

### 2.4 雪地履带车（T4 · 终极蓝图）

```
需求：金属×50 + 电子×20 + 马达×3 + 发电机核心 + 蓝图
速度：步行 ×1.5
货舱：200 格
乘员：全队 6 人
油耗：核燃料 0.05/km 或 燃油 5/km
维护：每次出行 → 电子×5 + 金属×8

特殊：无视天气封锁——暴风雪也能出行；车内可作为临时庇护所
```

## 三、世界地图规则变化

```
无载具：最大距离 3 km，燃料 = 距离 × 人数

有载具：
  · 燃料 = 载具油耗 × 距离 + 个人装备消耗
  · 最大距离由载具类型决定
  · 到达时间 = 距离 / 载具速度（搜刮窗口相对延长）
```

## 四、与搜刮场景的衔接

### 转移物资
搜刮中背包满 → 走到场景边缘 → 转移到载具货舱 → 清空背包继续搜刮。但每次转移消耗时间。

### 暴风雪庇护
卡车/履带车有车厢 → 暴风雪降临可躲入（温度 = 室外 +30°C）。摩托无车厢 → 仅 50% 遮风。

### 撤退
正常撤退：背包 + 货舱全部带回。紧急撤退：背包丢 30%，货舱不掉。

## 五、耐久与维护

```
耐久度 100 → 每次出行消耗 5-15

< 30%：速度 ×0.8，可能半路抛锚
< 10%：无法出行

维修（技师自动执行）：
  恢复 = 5 + 技师智力 / 游戏小时
  材料：木材/金属/电子元件（按载具类型）
```

## 七、阶段感

```
Day 1-15：步行——距离和背包是紧约束
Day 15-30：载具解锁——探索半径翻倍，约束变为燃料和维护成本
Day 30+：战略机动——履带车无视天气，从防御转向区域影响力
```

---

> 载具不是让游戏变简单，而是**改变约束的类型**。步行时代受限于距离，载具时代受限于燃料和维护。问题变了，决策也就变了。
